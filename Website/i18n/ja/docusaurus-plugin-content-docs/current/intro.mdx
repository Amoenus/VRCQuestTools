# はじめに

## VRCQuestTools

VRCQuestToolsはOculus Quest用にVRChatアバターをアップロードするためのエディタ拡張です。

- シングルクリックでPCアバターをQuest用にアップロード可能にします。
- 手動でアバターをQuest対応にするための様々なツールを使うことができます。

## 動作環境

- Unity 2019.4.31f1 または 2022.3.6f1
- VRChat SDK - Avatars 3.1.1 以降

## クイックガイド

### インストール

VRCQuestToolsをプロジェクトに追加するには、以下の手順に従ってください。

1. 下のボタンをクリックしてVCCにリポジトリを追加します。

  <p>
    <Link
      target={null}
      className="button button--primary button--lg"
      to="vcc://vpm/addRepo?url=https://kurotu.github.io/vpm-repos/vpm.json">
      <Translate>
        Add to VCC
      </Translate>
    </Link>
  </p>

2. VCCで管理しているプロジェクトにVRCQuestToolsパッケージを追加します。

### Quest用にアバターを変換

アバターをQuest用に変換するには、以下の手順に従ってください。

1. シーン内のヒエラルキーからアバターを右クリックします。
2. コンテキストメニューから**VRCQuestTools** > **Convert Avatar For Quest**を選択します。
3. 開いたウィンドウで**変換**ボタンをクリックします。
4. 変換が完了するのを待ちます。
5. 変換後のアバターをAndroidプラットフォーム用にアップロードします。

:::caution
このツールはパフォーマンスランクの最適化をしません。
多くの場合、変換後のアバターはQuest基準で **Very Poor** になります。

**Show Avatar** 操作をしない限り、フォールバックアバターで置き換えられます。
詳細については、[Quest Limits](https://docs.vrchat.com/docs/avatar-performance-ranking-system#quest-limits)
と [Avatar Fallback System](https://docs.vrchat.com/docs/avatar-fallback-system) を参照してください。
:::
